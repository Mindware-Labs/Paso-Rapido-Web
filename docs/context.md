You are working on a real-world frontend project for a Dominican Republic highway toll system called **Paso Rápido**.

---

## 🏢 PRODUCT CONTEXT

**Paso Rápido** is an electronic toll payment system (similar to EZ-Pass in the US).

Users:

* Attach a tag to their vehicle
* Recharge balance
* Pass through tolls without stopping

This landing page is meant to:

* Present the product
* Drive registrations
* Explain how it works
* Showcase the mobile app

---

## 🎯 PROJECT GOAL

We are redesigning the landing page as a **high-end interactive demo**, focused on:

* Visual impact
* Modern UX
* Smooth animations
* Premium feel

⚠️ Important:
This is NOT production backend work.
This is a **frontend-only showcase/demo**.

---

## 👤 TARGET USERS

* Drivers in Dominican Republic
* Logistics / fleet companies
* Everyday commuters

User mindset:

* Wants speed and simplicity
* Doesn’t want to wait in toll lines
* Needs trust and clarity

---

## 🎨 BRAND GUIDELINES

Primary color:

* Green (trust, speed, movement)
